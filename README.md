# 🎬 Amazon Prime Video — Exploratory Data Analysis (EDA)

> **A deep-dive Python EDA** into Amazon Prime Video's content library of 8,861+ titles, uncovering patterns in genres, ratings, regions, and content trends.

## 📌 Project Overview

This notebook performs a comprehensive Exploratory Data Analysis on the Amazon Prime Video dataset. It is part of a larger project that includes a Tableau dashboard and a live portfolio website.

## 💡 Key Findings

- **Drama, Comedy & Thriller** dominate content volume
- **Documentaries & Reality** shows earn higher audience ratings
- **TV shows** are significantly underrepresented vs movies
- **US, Canada & India** are the top content-producing countries
- Content is added in **bursts** with noticeable gaps
